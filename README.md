# C++ Learning Journey

This repo contains beginner C++ projects built while learning fundamentals through the Apna College C++ course.

## Projects

- **calculator/** - Choice-based calculator supporting sum, difference, product, division, and modulus, with input validation and divide-by-zero handling.
- **grade-calculator/** - Assigns a letter grade (A-F) based on marks input, with validation for out-of-range marks.
- **age-checker/** - Classifies a person as minor, adult, or senior citizen based on age input, with validation for negative ages.

## How to run any project

1. Navigate into the project folder (e.g. `cd calculator`)
2. Compile: `g++ main.cpp`
3. Run: `.\a.exe` (Windows) or `./a.exe` (Mac/Linux)

## What I'm learning

Variables, data types, input/output, operators, and conditional logic (if/else-if chains), with a focus on handling edge cases and invalid input properly rather than just getting the happy path working.
